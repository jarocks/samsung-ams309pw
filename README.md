# Samsung AMS309PW "Capella" Driver
A yet untested Kernel driver for the Samsung AMS309PW01/AMS309PW03 Super AMOLED Panel used in the BlackBerry Q10 and P'9983. Reverse engineered using a rooted Q10, Logic 8, Ghidra, and a lot of frustration.

A special thanks to [Oleksandr](https://github.com/bb10root) who's tooling and work achieving root on Blackberry 10 helped make this possible.
